# APCDemo
Sample project

## API's for the code

1) **Get optimal result** - Get's the optimal result for multiple saw mills
2) **Get max profit** - Private API. For getting the maximum profit for a given stream of logs
3) **Get all permuations** - Private API. Given a list of integers it will return all possible unique permuations
4) **Get profit for comibination** - Private API. Given a permutation, it will return the profit this perumation get can achieve with DEFAULT_CUTTING(3)
